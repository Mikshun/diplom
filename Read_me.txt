Описание проекта

Бот запускается из файла main.py,
Настроен на интеграции с API "The Fork The Spoon".
В данном апи собраны рестораны со всего мира, позваляя найти лучшие рестораны с фотографиями.

Логика бота разделена на пакеты:
    Handlers: основные комманды бота.
    Keyboard: Прикрепление клавиатуры, для пользователя.
    States: Настроенные состояния пользоватиля для реализации работы вопрос - ответ.
    Config-Data: В данной папке есть файл, для взятие переменных окружения.

В боте реалзованы такие команды как:
 /help — помощь по командам бота;
 /low — вывод самых лучших дешевых ресторанов;
 /high — вывод самых лучших дорогих ресторанов;
 /custom — вывод ресторанов по запросам пользователя;
 /history — вывод истории запросов пользователей.

Скрины команд и работы можно увидеть в папке exmpl_images.
Если по городу нет информации, бот даст соответствующий ответ.
